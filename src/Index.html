Introducción

Este sistema permite gestionar el inventario de productos, registrar ventas y generar reportes de forma eficiente para el salón de belleza JMC.

Funcionalidades

  • Gestión de productos y control de stock.
  • Registro de ventas diarias.
  • Generación de informes de inventario y ventas.
  • Interfaz amigable y fácil de usar.
