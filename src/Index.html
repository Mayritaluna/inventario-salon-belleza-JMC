Inicio

Bienvenido al sistema de gestión de inventario del salón de belleza JMC. Aquí podrás gestionar todo el inventario y las ventas de manera eficiente.

Inventario

En esta sección podrás ver y gestionar todos los productos disponibles en el salón de belleza. Puedes añadir, eliminar o modificar productos en el inventario.

Ventas

Aquí puedes registrar las ventas realizadas en el salón. Lleva un control de las ventas diarias y visualiza estadísticas.

Reportes

Genera reportes sobre el inventario y las ventas. Puedes ver informes detallados de los productos vendidos y el estado del inventario.

Contacto

Para más información, contáctanos a través de nuestro correo electrónico: info@salonbellezaJMC.com.

Introducción

Este sistema permite gestionar el inventario de productos, registrar ventas y generar reportes de forma eficiente para el salón de belleza JMC.

Funcionalidades

  • Gestión de productos y control de stock.
  • Registro de ventas diarias.
  • Generación de informes de inventario y ventas.
  • Interfaz amigable y fácil de usar.
